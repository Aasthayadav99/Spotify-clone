## 📖 About The Project

This project is a faithful frontend recreation of Spotify's web player UI. The goal was to deeply understand **CSS layout systems**, **design consistency**, and **component-based thinking** — all without using any JavaScript framework or CSS library.

Every section — the sidebar, top navigation, playlist cards, music player bar — was hand-crafted using **Flexbox** and careful attention to Spotify's actual design language.

## ✨ Features

- 🎨 **Faithful UI recreation** — closely mirrors Spotify's actual design
- 📐 **Flexbox layout** — sidebar, content area, and player bar built with Flexbox
- 🎵 **Music player bar** — fixed bottom player with progress bar and controls
- 📁 **Sidebar navigation** — Home, Search, and Your Library sections
- 🃏 **Playlist cards** — grid of album/playlist cards with hover effects
- 📱 **Responsive design** — adapts across different screen sizes
- 🌑 **Dark theme** — Spotify's signature dark color palette

## 🚀 Getting Started

### Prerequisites

Just a modern web browser!

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Aasthayadav99/Spotify-clone.git

# Navigate to the folder
cd Spotify-clone

# Open in browser
open index.html
```
